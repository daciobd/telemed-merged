Página Pública do Médico

Para acessar o perfil público de um médico e agendar uma consulta diretamente, use a URL no formato:

https://seu-dominio.com/dr/NOME-DO-MEDICO

Exemplo:

/dr/dr-joao-silva

Como funciona?

Cada médico cadastrado no TeleMed pode ter uma URL personalizada. Os pacientes podem usar esse link para ver o perfil público, consultar a agenda disponível e agendar uma consulta diretamente.

Sou médico, como configuro minha URL?

Acesse o Consultório Virtual, vá em "Consultório Virtual" no menu e configure sua URL personalizada.
