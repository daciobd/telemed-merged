Versão leiga • 1 página

Paciente — Passo a passo

  • Entrar: Começar Demo → Paciente (use nome fictício).
  • Minha oferta: escolha um valor. Sem médico agora? clique Ver slots e agende.
  • Sala de espera: Abrir chat, envie mensagem e, se quiser, um arquivo (PDF/foto, até 10MB).
  • Entrar no vídeo: quando o médico abrir o consultório, o botão Entrar habilita.
  • Se travar: mude para Áudio e continue pelo chat.
  • Depois: veja o PHR → Prescrições recentes. Use Ver PDF (ou Reimprimir link se expirado).

Médico — Passo a passo

  • Entrar: Começar Demo → Médico.
  • Dashboard: na Fila, Aceitar para atender agora ou faça Contraoferta. Sem agenda? Ver slots.
  • Consulta: Abrir consultório, Chat para avisar e Convidar o paciente.
  • Registro: preencha Queixa/HDA e busque CID‑10/CIAP em Hipótese.
  • Prescrição: Nova Prescrição → busca ANVISA → itens/posologia → Emitir. Pode Reimprimir link.
  • Finalizar: Finalizar e responder NPS. No histórico, aparece Receita emitida.
Dicas rápidas: Use nomes/documentos de teste. Se não achar algo, procure o botão com o nome da ação (ex.: Ver slots, Nova Prescrição). Em dúvida, clique no ? Ajuda.

Cenários para validar

  • Sem médico imediato → Ver slots e agendar ✓
  • Rede ruim → alternar para Áudio + usar Chat ✓
  • Paciente envia arquivo na sala de espera ✓
  • CID/CIAP: buscar por nome e por código ✓
  • Emitir RX → Ver PDF → Reimprimir link (se expirado) ✓

Estados de verificação da receita

  • VÁLIDA: assinatura confere
  • EXPIRADA: gerar link novo (reimprimir)
  • REVOGADA: não dispensar
Privacidade: use dados fictícios. Arquivos são só para este teste.
